What Is Bullet It?

Bullet It is an an organisational application whose design has been inspired by the immensely popular anolog system called Bullet Journaling. "So what is Bullet Journaling?" you may as. Created by New York-based designer Ryder Carroll, the bullet journal is an organizational notebook system that can keep track of anything you want in a coherent and easy-to-manage way. It can be a to-do list, an event planner, an idea catcher, or whatever else you can think of. In fact, bullet journals are often all-of-the above. You can find oout more about its origins here

How Does It Work?

Bullet It allows the user to create "collections" of tasks, events or notes that they may come across in their daily lives and store them in this easy to use application.

Each item that the user records is tagged with a "bullet":

Denotes an To-do

Denotes an Event

Denotes a Note

The following bullets can also be added:

Denotes that something is Done

Denotes that something is Important

Bullet it also allows the user to assign these snippets of information to a future month and displays these clearly in the user's "Future Log".

Here an item can be deferred to another month by hitting the Migrate button
